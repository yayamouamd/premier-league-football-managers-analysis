# Premier League Managers Analysis

## Overview
This project analyzes 20 Premier League managers using Python and data visualization techniques.

## Objectives
- Compare managers' performance.
- Explore goals, wins, and possession statistics.
- Identify tactical trends and playing styles.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
Statistics for 20 Premier League managers.
